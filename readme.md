# 🍬 Candy Castle Collector

A mobile browser game commissioned and developed as a promotional tool for a local retail shop in Egypt. Built solo using **Unity (WebGL)** and deployed as a web app.

---

## 🎮 Gameplay

Players control a barrel at the bottom of the screen and catch falling candy items. Each item affects the score differently:

- ✅ **Good candy** — collect to gain points and earn lives
- ❌ **Rotten/bad candy** — avoid or lose points

The higher your final score, the better the **discount reward** you unlock at the end of the game.

---

## 🎁 Discount & Anti-Abuse System

At the end of each game, the player receives a discount coupon tied to their score. The coupon includes:

- The **discount percentage** earned based on performance
- A **date and timestamp** to prevent reuse or screenshot abuse
- A message prompting the player to show the screen to the cashier

This was a custom business requirement — the timestamp system was designed and implemented entirely by me to protect the client from coupon fraud.

---

## 🛠️ Tech Stack

- **Engine:** Unity
- **Platform:** WebGL (mobile browser)
- **Language:** C#
- **Deployment:** Web app (`castle-game.web.app`)
